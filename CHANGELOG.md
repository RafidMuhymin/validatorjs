### [1.0.0](https://github.com/chantouchsek/validator-js/compare/v0.0.7...v1.0.0) (2022-01-01)

### Breaking Changes

- From

```js
const validator = new Validator({
  input,
  rules,
  customMessage: {},
  customAttributes: {},
  locale: 'en',
  confirmedReverse: true,
})
```

- to

```js
const options = {
  customMessage: {},
  customAttributes: {},
  locale: 'en',
  confirmedReverse: true,
}
const validator = new Validator(input, rules, options)
```
